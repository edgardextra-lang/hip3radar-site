Public log of every meaningful release. Build-in-public, in chronological order, newest first.

v0.4 — Trust infrastructure

2026-04-18 major
  • New design system. Bloomberg-terminal aesthetic. Geist Mono throughout. Green accent (#7ef3a0) replaces pink.
  • Per-deployer scorecard pages at /deployer/<slug> — aggregate grade, market table, embed snippet, "Get Verified" CTA.
  • Per-market detail pages at /market/<dex>/<coin> — full scorecard, 24h history chart, signal breakdown.
  • Embeddable badge widget via /badge.js — drop the <hip3radar-badge> element on any market page.
  • Letter grades (AA / A / B+ / B / C / D / F) computed from composite risk score on every market.
  • Verified Deployer tier ($2,500+/mo) replaces generic Enterprise — badge embed rights, custom oracle integration, dedicated incident response.
  • Status page at /status — live health for polling, bot, API, tunnel, email.
  • OpenAPI 3.1 spec at /openapi.json. Interactive docs at /docs.
  • Animated GIF favicon with radar sweep.
  • Search + dex filter on the live scoreboard.
  • Pricing rebalanced: Pro $29 → $49, Team $199 → $499, Enterprise → Verified Deployer $2,500+.
  • Sticky bottom Telegram CTA removed (moved to hero).

v0.3 — Telegram alerts

2026-04-17 minor
  • Telegram alert bot live at @hip3radar_alerts_bot — multi-user, custom thresholds, dex filter, 6h dedup per market.
  • Slash commands: /start, /subscribe, /list, /unsubscribe, /threshold, /dex, /help.
  • Inline-button onboarding (Warning 40+ / Critical 60+ presets).
  • Free during public beta. Telegram chat IDs stored in SQLite, deduped on insert.
  • Mobile optimization across all pages (480px + 360px breakpoints, 44px tap targets).

v0.2 — Distribution + SEO

2026-04-16 minor
  • Domain hip3radar.xyz live via named Cloudflare Tunnel with auto-TLS.
  • Custom domain email on hip3radar.xyz with SPF + DKIM + DMARC alignment via Resend + Cloudflare Email Routing.
  • Dual-mode confirmation emails: instant plain-text "you're on the list" + scheduled rich HTML follow-up.
  • llms.txt + llms-full.txt for LLM agent discovery.
  • Dynamic sitemap.xml (330+ URLs including every market and incident).
  • JSON-LD structured data (Organization, SoftwareApplication, FAQPage) for rich search results.
  • Explicit allow rules for GPTBot, ChatGPT-User, ClaudeBot, PerplexityBot, etc.
  • OG link preview cards across iMessage / Telegram / Slack / Discord / Twitter.
